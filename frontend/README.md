# Clawbros Frontend

This is the frontend dashboard for the Claw Swarm project.

## Tech Stack
- **React** (Vite)
- **Tailwind CSS** (Styling)

## Getting Started
1. `npm install`
2. `npm run dev`
3. Access at `http://localhost:5173`

## Features
- Displays agent status cards (mocked for now, will connect to backend `/api/agents`).
